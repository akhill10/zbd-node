## Introduction

This is the SDK for NodeJS that interacts with the ZEBEDEE APIs.

## Installation

If you're an `npm` user:

```bash
npm install @zbd/node
```

Or if you're a `Yarn` user:

```bash
npm install @zbd/node
```
